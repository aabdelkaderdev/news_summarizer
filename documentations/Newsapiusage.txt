If you want headlines just from a specific source, for example BBC News, we can do that too.

The identifier for BBC News is bbc-news, which we can get by querying the sources endpoint.
Definition

GET https://newsapi.org/v2/top-headlines?sources=bbc-news&apiKey=API_KEY

Example request

import requests
url = ('https://newsapi.org/v2/top-headlines?'
       'sources=bbc-news&'
       'apiKey=1322bde53c4146b395c4dd4c6e15189f')
response = requests.get(url)
print response.json()
